Setup: GUI client

If you want to install your own About the Option C steps:

  1. Download the client source from GitHub https://github.com/lupyanlab/Rule-Game
  2. Install the Node.js programming language https://nodejs.org/en/ (preferably LTS version)
  3. Run “npm install” from the top directory
  4. Run “npm run build” from the top directory
  5. Copy all of the files inside the generated folder called “build/” to where you are hosting static files using a web server. For example, 
        mkdir /opt/tomcat/webapps/game-data
    cp -pa build /opt/tomcat/webapps/game-data/prod

If you know TypeScript, you can modify the client's source code as needed before compiling.
